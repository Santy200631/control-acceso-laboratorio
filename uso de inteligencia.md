# Uso de Inteligencia Artificial

##  Descripción general

Se utilizó inteligencia artificial como herramienta de apoyo para comprender conceptos, estructurar el proyecto y mejorar la calidad del código, sin copiar soluciones completas.

---

##  Prompts utilizados

### Prompt 1:

"Cómo estructurar un proyecto en Java con arquitectura por capas"

**Uso:**
Se utilizó para entender la separación entre entidades, acceso a datos, lógica de negocio y presentación.

**Ajustes realizados:**
Se adaptó la estructura al enunciado del proyecto y a los requisitos del profesor.

---

### Prompt 2:

"Cómo guardar y leer archivos .txt en Java"

**Uso:**
Se utilizó para implementar la persistencia de datos en archivos.

**Ajustes realizados:**
Se integró con las clases DAO del proyecto.

---

### Prompt 3:

"Cómo validar datos en Java (evitar duplicados)"

**Uso:**
Se aplicó en la validación de usuarios con ID repetido.

**Ajustes realizados:**
Se adaptó a listas obtenidas desde archivos.

---

### Prompt 4:

"Cómo calcular diferencia entre fechas en Java"

**Uso:**
Se utilizó para calcular el tiempo total dentro del laboratorio.

**Ajustes realizados:**
Se implementó usando LocalDateTime y Duration.

---

##  Justificación técnica

El uso de IA permitió:

* Comprender mejor conceptos de arquitectura en capas
* Optimizar el tiempo de desarrollo
* Mejorar la calidad del código
* Aplicar buenas prácticas

Sin embargo, todo el código fue revisado, adaptado y probado manualmente.

---

##  Conclusión

La inteligencia artificial fue utilizada como apoyo educativo y no como sustituto del aprendizaje, cumpliendo con las normas establecidas.
